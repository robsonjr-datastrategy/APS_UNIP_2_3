# APS UNIP 2/3 - Calculo de Emergia

Projeto desenvolvido para a APS da UNIP com o objetivo de simular o calculo de emergia em uma rede de processos.

O sistema foi feito em Python e utiliza uma estrutura de grafo para representar processos, conexoes e fluxos. A partir dos dados informados pelo usuario, o programa calcula a emergia propagada entre os processos usando busca em profundidade (DFS) e gera um relatorio com os resultados.

## Funcionalidades

- Cadastro de processos
- Cadastro de conexoes entre processos
- Definicao do parametro `minflow`
- Calculo de emergia
- Geracao de relatorio em `.txt`
- Consulta da fundamentacao teorica pelo terminal

## Como executar

```bash
python main.py
```
